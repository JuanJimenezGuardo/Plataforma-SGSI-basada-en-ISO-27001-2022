# z_docs - Indice maestro

Este directorio concentra la documentacion estrategica, tecnica y de presentacion del proyecto VIT.

## Ruta rapida para reunion (10-15 min)
1. Overview: `00_overview/`
2. Arquitectura principal: `01_architecture/00_principales/`
3. Estado de sprints: `02_sprints/`
4. Evidencia tecnica backend: `03_engineering/backend/`
5. Material para presentacion: `04_presentacion/`

## Estructura
- `00_overview/`: contexto de negocio y resumen para desarrollo.
- `01_architecture/`: arquitectura formal, organizada por prioridad.
- `02_sprints/`: plan general y estado por sprint.
- `03_engineering/`: evidencia de implementacion tecnica.
- `04_presentacion/`: material preparado para reunion/exposicion.

## Recomendacion de uso
Si hay poco tiempo, revisar en este orden:
1) `01_architecture/00_principales/RESUMEN_EJECUTIVO.md`
2) `01_architecture/00_principales/FLUJO_SGSI.md`
3) `02_sprints/sprint_2/ESTADO_BACKEND_ACTUAL.md`
4) `04_presentacion/01_reunion/INFORME_PROGRESO_ACTUAL_PROFESOR.md`
